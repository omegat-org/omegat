OmegaT 5.8.0

Erste Schritte

Drücken Sie F1 oder rufen Sie das Menü > Hilfe auf, um die Bedienungsanleitung zu öffnen. Sie enthält eine ausführliche Einführung in OmegaT. Um OmegaT mit wenigen Einstellungen zu verwenden, gehen Sie wie folgt vor:

  1. Verwenden Sie > Projekt > Neu..., um ein neues Projekt zu erstellen.

  2. Ändern Sie die Sprachcodes, um Ihre Quell- und Zielsprache anzugeben. Klicken Sie auf OK.

    → OmegaT wird einen Ordner erstellen, der Ihre Dateien enthält. Außerdem wird ein leerer Übersetzungsspeicher ("translation memory" oder kurz TM) erstellt, der im Hintergrund gefüllt wird, ein Segment nach dem anderen, während Sie Ihre Übersetzung eingeben. Übereinstimmungen zwischen dem aktiven Segment und diesem TM werden im Fuzzy Matches-Bereich angezeigt.

  3. Klicken Sie auf Dateien hinzufügen... und fügen Sie die Dateien hinzu, die Sie übersetzen möchten. Schließen Sie das Fenster.

    → OmegaT wird die ausgewählten Dateien in das Projekt kopieren, das Sie im vorherigen Schritt erstellt haben. Sie werden nun segmentweise angezeigt, damit Sie sie übersetzen können.

  4. Tragen Sie die Übersetzung des ersten Segments ein. Drücken Sie Enter.

    → OmegaT wird dieses Segment in den TM des Projekts aufnehmen und den Textcursor zum nächsten Segment bewegen.

  5. Mit > Projekt > Übersetzte Dateien erstellen können Sie das übersetzte Dokument jederzeit erstellen.

    → OmegaT wird die übersetzten Dateien im target-Ordner des Projekts ablegen, das Sie im ersten Schritt erstellt haben. Verwenden Sie > Projekt > Auf Projektinhalte zugreifen, um auf die Dateien zuzugreifen.

  6. Wenn Sie mehr wissen wollen, lesen Sie die Bedienungsanleitung.

  • Die Freiheit, die Software für beliebige Zwecke zu nutzen.

  • Die Freiheit, die Funktionsweise von OmegaT zu analysieren und es so zu verändern, dass es nach Ihren Wünschen läuft.

  • Die Freiheit, Kopien weiterzugeben, damit Sie anderen helfen können.

  • Die Freiheit, Kopien der von Ihnen geänderten Versionen an andere weiterzugeben.
