OmegaT 6.0.0 - Bedienungsanleitung

Zusammenfassung

Dieses Dokument ist die offizielle Bedienungsanleitung für OmegaT, ein kostenloses Tool für computergestützte Übersetzung.

Der Hauptzweck von OmegaT besteht darin, Ihnen einen möglichst zuverlässigen Zugriff auf Ihre Übersetzungsressourcen zu ermöglichen, damit Sie so schnell wie möglich eine gute Übersetzung anfertigen können.

OmegaT wird unter den Bedingungen der GNU General Public License, Version 3.0 verteilt. Diese Lizenz gibt Ihnen die folgenden Freiheiten:

  • Die Freiheit, die Software für jeden Zweck zu verwenden.

  • Die Freiheit, die Funktionsweise von OmegaT zu analysieren und es so zu verändern, dass es Ihren Wünschen entspricht.

  • Die Freiheit, Kopien weiterzugeben, um anderen zu helfen.

  • Die Freiheit, Kopien der von Ihnen geänderten Versionen an andere weiterzugeben.
