Zusammenfassung

Dieses Dokument ist die offizielle Bedienungsanleitung für OmegaT, dem kostenlosen Tool für computergestützte Übersetzung.

Der Hauptzweck von OmegaT besteht darin, Ihnen auf möglichst zuverlässige Weise Zugang zu Ihren Übersetzungsressourcen zu verschaffen, damit Sie eine gute Übersetzung so schnell wie möglich erstellen können.

OmegaT wird unter den Bedingungen der GNU General Public License, Version 3.0 verteilt. Die Lizenz gewährt Ihnen die folgenden Freiheiten:

  • Die Freiheit, die Software für beliebige Zwecke zu nutzen.

  • Die Freiheit, die Funktionsweise von OmegaT zu analysieren und es so zu verändern, dass es nach Ihren Wünschen läuft.

  • Die Freiheit, Kopien weiterzugeben, damit Sie anderen helfen können.

  • Die Freiheit, Kopien der von Ihnen geänderten Versionen an andere weiterzugeben.
