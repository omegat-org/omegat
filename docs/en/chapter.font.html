1. Font settings

In this dialog one can define the font used by OmegaT in the following windows:

  • OmegaT main window (except the Dictionary viewer)

  • Search window

  • Tag validation window

The dialog can be accessed via the item in the application menu. The dialog contains: Options â†’ Font...

  • Font. Drop-down to select one of the fonts available on your machine.

  • Size. Edit to change font size.

  • Sample text. Immediate preview of the selected font.

Note. The display update can take more time if the number of displayed segments is important. Also, some fonts may have better coverage of your language pair.
