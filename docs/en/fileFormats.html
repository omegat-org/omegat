Supported File Formats

OmegaT 1.6.2 natively supports translating the following types of files:

  • Microsoft Open XML files
  • OpenDocument files (also known as OpenOffice.org or StarOffice documents)
  • HTML and well-formed XHTML
  • DocBook files
  • PO files
  • Plain text files
  • Java localization files (resource bundles)
  • INI files (Key=Value text)
  • HTML Help Compiler files

With a bit of hand-work you may also translate Microsoft Office files (other than those saved in the Microsoft Open XML format). Simply use OpenOffice.org to convert your Microsoft Office files to and from corresponding OpenDocuments. For Microsoft Word files another way is to save them to HTML using Microsoft Word itself.
