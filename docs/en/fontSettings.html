Font Settings

In this dialog you may modify the font OmegaT uses to display source, translation, matches and glossary terms. It is called through the Options -> Font... main menu item.

The dialog contains:

  • Drop-down to select one of the fonts available on your machine.
  • Spin edit to change font size.
  • An area with the same text to preview the selected font immediately.

Warning! It may take a noticeable amount of time for OmegaT to update the display after you have changed the font setting, especially when a big file with many segments is open in the editor. Also, some fonts behave better for some language pairs than for others. Generally speaking, it is better to change of font before opening a project. The penalty is most painful on G4 + Mac OS X + Japanese combination.
