Requisitos del sistema

OmegaT requiere el entorno de ejecución Java (Java Runtime Environment). Básicamente, si su sistema puede ejecutar el entorno de ejecución Java (también conocido como J2RE), podrá ejecutar OmegaT. El entorno de ejecución Java está disponible para las siguientes plataformas: Windows 95/98/2000/ME/XP, UNIX, Linux, Solaris (x86 y Sun SPARC) y Mac OS 9. Viene preinstalado en el Macintosh OS X.

Hasta donde sabemos, OmegaT no puede ejecutarse en Windows NT debido a un error en su implementación de Java.
