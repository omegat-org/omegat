OmegaT-ren instalazioa

OmegaT instalatzeko, destrinkotu deskargatutako fitxategia. Horrela, omegat/ direktorio bat sortuko da, eta bertan, OmegaT erabiltzeko beharrezkoak diren fitxategi guztiak egongo dira.

.tar.gz fitxategia destrinkotzeko:

$ tar xf deskargatutako_fitxategia.tar.gz

OmegaT menuetara (KDE) edo paneletara (GNOME) gehitzea

KDE erabiltzaileak

OmegaT zuen menuari gehitzeko, egin ondoko hau:

  • Kontrol gunea - Mahaigaina - Panelak - Menuak - Editatu K menua - Fitxategia - Elementu berria/Berria
  • Gero, menu egoki bat hautatu ondoren, gehitu azpimenu/elementu bat Fitxategia - Azpimenu berria eta Fitxategia - Elementu berria erabiliz. Sartu OmegaT elementu berriaren izen gisa.
  • "Komandoa" eremuan, erabili nabigazio-botoia OmegaT-ren abioko script-a bilatzeko (alegia, destrinkotutako direktorioaren barruan OmegaT deitzen den fitxategia), eta hautatu ezazue.
  • Klikatu ikono-botoian (Izena/Deskribapena/Iruzkinak eremuen eskuinaldean) - Beste ikonoak - Arakatu, eta joan OmegaT-ren /images azpikarpetara. Hautatu OmegaT.png ikonoa.
  • Azkenik, gorde aldaketak Fitxategia - Gorde aukera erabiliz.

GNOME erabiltzaileak

OmegaT zuen panelari (pantailaren goialdean dagoen barrari alegia) gehitzeko, egin ondoko hau:

  • Eskuineko botoiarekin klikatu Panelari gehitu - Abiarazle berezitua.
  • "Izena" eremuan, idatzi "OmegaT" ; "Komandoa" eremuan, erabili arakatze-botoia OmegaT-ren abioko script-a bilatzeko (alegia, destrinkotutako direktorioaren barruan OmegaT deitzen den fitxategia). Hautatu script-a eta onartu OK sakatuz.

OmegaT abian jartzea

OmegaT instalatu ondoren, komando-lerrotik abiarazi daiteke, komando-lerrotik abiarazteko beharrezkoak diren parametroak dituen script bat sor daiteke, edo zuzenean abiarazi daiteke OmegaT.jar fitxategian klikatuz. Metodoa bata edo bestea izango da banaketaren arabera. Segurtatu zure sistemako PATH ezarpena zuzena dela eta .jar fitxategiak Java abiarazle bati lotuta daudela.

OmegaT hasierako menura edo mahaigainera gehitzea

OmegaT-JRE.exe, OmegaT.exe edo OmegaT.bat fitxategietako bat mahaigainera edo hasiera menura arrastatu daiteke aplikazioa bertatik abiarazteko.

OmegaT abian jartzea

OmegaT instalatu ondoren, OmegaT.jar fitxategian klikatuz abiarazi daiteke, OmegaT.bat fitxategian klikatuz ere abiarazi daiteke (fitxategi honetan komando-lerroko parametroak gehi daitezke) edo zuzenean komando-lerrotik abiarazi daiteke.

Beste sistema batzuk

Deskargatu beharreko paketea

Linux (Intel x86) eta Windows plataformetarako, OmegaT-ren paketean Sun Java JRE bat gehitu da. Beste plataforma batzuetan (Linux PowerPC, Linux x64, Solaris SPARC/x86/x64, Windows x64, etab.), Java JRE bateragarri bat eduki behar da sisteman instalatuta OmegaT erabilil ahal izateko.

→ Bai
deskargatu OmegaT_1.7.3_Without_JRE.zip
*Pakete hau Java 1.4 JRE-rekin bateragarria den JRE-a instalatuta daukan edozein plataformatan erabili daiteke.
→Ez dakit
ireki terminal bat eta idatzi "jva -version". "command not found" edo antzeko mezuren bat jasotzen bada, litekeena da Java ez dagoela instalatuta.
→Ez
eskuratu zure sistemarako Java JRE bat (ikus behean) eta deskargatu OmegaT_1.7.3_Without_JRE.zip.

SUN-ek JRE-ak eskaintzen ditu Solaris SPARC/x86 (Java 1.4) eta Linux x64, Solaris x64, Windows x64 (Java 1.5) sistemetarako:
http://java.sun.com/products/archive/
IBM-k JRE-ak eskaintzen ditu Linux PowerPC-rako:
http://www.ibm.com/developerworks/java/jdk/linux/download.html

Jarraitu zure sistemak behar duen paketearen instalaziorako argibideak.

Komando-lerroaren bidezko abioa

Normalean, ez da beharrezkoa OmegaT komando-lerrotik abiaraztea. Hala ere, komando-lerroaren metodoa erabiltzen bada OmegaT abiarazteko, aplikazioaren portaeraren zenbait alderdi aldatu eta moldatu daitezke. Bi modu daude OmegaT komando-lerroz abiarazteko.

OmegaT komando-lerrotik kontrolatu nahi duten Mac OSX paketearen erabiltzaileei "Beste sistema batzuk" atalean eskuragarri dagoen paketea erabiltzea aholkatzen zaie. Ikus goian.

1. metodoa: komando-lerroko leihoa irekitzea

Komando-lerroko leihoari "terminala" ere deitzen zaio. Windows-en "MS-DOS leiho" deritzo, eta Hasiera menuan dago, Aplikazioak barruan, "MS-DOS" ikonoaren bidez. Mac OSX-erako pareko elementua Applications → Utilities atalean dagoen Terminal aplikazioa da.

OmegaT abiarazteko, normalean bi komando idatzi behar dira. Lehena hau da:

cd {karpeta}

non {karpeta} OmegaT aplikazioa dagoen karpeta den, bide-izen eta guzti - zehatzago esateko, OmegaT.jar dagoen lekua da. Praktikan, komando hau honelako zerbait izango da:

Windows-en
cd C:\Program Files\OmegaT
Mac OSX-en
cd /Applications/OmegaT
Linux-en
cd /usr/local/omegat

Komando hauek OmegaT fitxategi exekutagarria daukan karpetara joateko balio dute. Bigarren komandoa OmegaT abiarazten duena da. Oinarrizko forman, komandoa hau da:

java -jar OmegaT.jar

Kontuan izan maiuskulak, garrantzitsuak dira.

Metodo honi esker errazagoa da akatsak aurkitzea: aplikazioaren erabileran arazoren bat gertatzen bada, informazio erabilgarria eman dezakeen akats-mezu bat agertzen da terminalean.

2. metodoa: .bat fitxategia/abioko script-a aldatzea

1. metodoa ez da aplikazio bat sarritan abian jartzeko modurik praktikoena. Hori dela-eta, goian deskribatu diren bi komandoak fitxategi batean gorde dira (zehatzago izateko, "script" batean, Windows sistemetan ".bat fitxategi" ere deitzen dena).

Fitxategi hau exekutatzen denean, barruan dauden komandoak automatikoki exekutatzen dira. Beraz, abio-modua aldatzeko, aski da fitxategia aldatzea. OmegaT fitxategitik abiarazten denean, komando berriak exekutatu egingo dira.

Abiorako komandoaren argumentuak

Oinarrizko komandoak goian aipatu dira. Komando hauek aldatzeko, "argumentuak" erantsi behar zaizkie. Argumentuak hasierako "java" ondoren eta "-jar OmegaT.jar" baino lehen gehitu behar dira.

Ondoren argumentu posibleen zerrenda ematen da. Argumentuei buruzko informazio gehiago eduki nahi izanez gero, idatzi man java terminalean. man komandoaren bidez, hautatutako komandoaren eskuliburua erakusten da. Gerta daiteke zure sistemak ez edukitzea java komandorako eskulibururik zure hizkuntzan.

-Duser.language=XX

Normalean, hau da, OmegaT argumenturik gabe abiarazten denean, aplikazioak erabiltzailearen sistema eragilearen hizkuntza detektatzen du. OmegaT-k hizkuntza horretan dagoen erabiltzaile-interfazerik badauka, OmegaT-k horixe erabiltzen du. Hortaz, erabiltzailearen sistema eragilea errusieraz badago eta OmegaT errusieraz lokalizatu bada, OmegaT-ren erabiltzailearen interfazea, menuak, etab. errusieraz erakutsiko dira. Erabiltzailearen sistemaren hizkuntza ezin bada eskuratu, OmegaT-ren hizkuntza lehenetsia ingelesa da. Hau portaera lehenetsia da.

"-Duser.language=XX" argumentua erabiliz gero, OmegaT-k sistema eragilearena ez den beste hizkuntza bat erabili dezan lortu daiteke. "XX" jartzen duen lekuan, erabili nahi den hizkuntzaren bi digituko kodea idatzi behar da. Esaterako, OmegaT euskarazko interfazearekin abiarazi nahi izanez gero (adibidez, sistema eragilea gaztelaniaz badago), komandoa hau izango litzateke:

java -Duser.language=eu -jar OmegaT.jar

-Duser.country=XX

Hizkuntzarekin batera, herrialdea ere zehaztu daiteke, esaterako CN edo TW txineraren kasuan. Hasiera azkar zuzena ireki dadin, hizkuntza eta herrialdea, biak zehaztu behar dira. Hau ezinbestekoa da konbinazio bakarra existitzen den kasuetan ere, esaterako pt_BR Brasilgo portugesaren kasuan. .

-Dswing.aatext=true

Argumentu honek letra-tipoak leuntzen ditu, beren itxura hobetuz.

-XmxZZM

Komando honek memoria gehiago esleitzen dio OmegaT-ri. Portaera lehenetsi gisa, OmegaT-k 256 MB-eko memoria dauka; beraz, hori baino gutxiago esleitzen bazaio ez da inolako abantailarik lortuko. "ZZ" jartzen duen lekuan, OmegaT-ri esleitu nahi zaion memoria-kantitate berria. megabyte-tan, idatzi behar da. Esaterako, OmegaT 1.024 MB-eko (1 GB) memoriarekin abiarazi nahi izanez gero, ondoko hau idatzi behar da:

java -Xmx1024M -jar OmegaT.jar

Argumentuak konbinatu egin daitezke. OmegaT goian aipatu diren adibide guztiekin abiarazi nahi izanez gero, komandoa ondoko hau izango litzateke:

java -Dswing.aatext=true -Duser.language=pt -Duser.country=BR -Xmx1024M -jar OmegaT.jar

OmegaT iturburutik eraikitzea

Deskargatu beharreko paketea

Deskargatu OmegaT_1.7.3_Source.zip
Pakete honek bertsio honi dagozkion fitxategien iturburu-fitxategiak dauzka eta Java 1.4 JDK edo bertsio bateragarri bat instalatuta daukan edozein sistematan erabili daiteke. OmegaT eraikitzeko, ant izeneko utilitatea ere beharko duzu.
Koderik berriena ere eskuratu dezakezu CVS zerbitzaritik:
cvs -z3 -d:pserver:anonymous@omegat.cvs.sourceforge.net:/cvsroot/omegat co -P omegat

OmegaT-ren eraikitzea

Destrinkotu OmegaT_1.7.3_Source.zip fitxategia eta sartu OmegaT_1.7.3_Source direktorioan, edo sartu CVS-tik deskargatutako kodearen /omegat/ direktorioan. Egiaztatu direktorio horretan build.xml fitxategi bat dagoela. Orduan, idatzi komando-lerroan:

$ ant jar release

Horrela, OmegaT-ren banaketa oso bat sortuko da ./dist/ direktorioan, eta bertan, OmegaT erabiltzeko beharrezkoak diren fitxategi guztiak egongo dira.
