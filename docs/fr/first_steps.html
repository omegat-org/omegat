OmegaT 6.0.1

Premiers pas

Appuyer sur la touche F1 ou accéder au menu Aide pour ouvrir le manuel d’utilisation. Ce manuel fournit une introduction complète à OmegaT. Pour utiliser OmegaT avec des paramètres simplifiés, procédez comme suit :

  1. Dans le menu Projet, sélectionnez Nouveau… pour créer un nouveau projet.

  2. Indiquez la langue du fichier source ainsi que la langue de traduction. Cliquez sur OK.

    → OmegaT créera un ensemble de dossiers contenant vos fichiers. Il créera également une mémoire de traduction vide qu’il remplira en arrière-plan, un segment à la fois, au fur et à mesure de votre traduction. Les correspondances pour cette mémoire apparaitront éventuellement dans le volet Correspondances.

  3. Cliquez sur Ajouter des fichiers… et ajoutez les fichiers que vous souhaitez traduire. Fermez la fenêtre.

    → OmegaT copiera les fichiers sélectionnés dans le projet créé lors de l’étape précédente. Ils sont affichés segment par segment pour que vous puissiez les traduire.

  4. Saisissez la traduction du premier segment. Appuyez sur Entrée.

    → OmegaT enregistre ce segment dans la mémoire de traduction du projet et déplace le curseur vers le segment suivant.

  5. Pour créer le document traduit, allez dans >Projet>Créer les documents traduits.

    → OmegaT conservera les fichiers traduits dans le dossier cible du projet créé lors de la première étape. Pour accéder aux fichiers, utilisez >Projet >Accéder au contenu du projet.

  6. Pour en savoir plus, consultez le manuel d’utilisation.
