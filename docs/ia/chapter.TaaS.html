1. Generalitates

Le servicio TaaS al sito https://demo.taas-project.eu/info supple servicios de terminologia in le linguas europee (plus russo). Illo permitte accesso sia al datos public e private, ubi le glossarios private (denominate "collectiones") pote esser extrahite ex le documentos existente, e le terminos de destination in parte populate automaticamente per fontes varie.

2. Crear un clave

Pro acceder le servicio TaaS, un usator debe crear un clave per https://term.tilde.com/account/keys/crear?systema=OmegaT.

Le clave debe alora esser date a OmegaT per -Dtaas.user.key=xxxxx. Le lanceatores del configuration de OmegaT (OmegaT.l4J.ini, omegat.kaptn e Configuration.properties) contine un entrata de patrono.

3. Accesso al servicio TaaS

Clicca sur Optiones, glossario pro monstrar le optiones sequente:

Navigar per le collectiones TaaS permittera le navigation inter le existente collectiones pro le linguas fonte e destination del projecto, e discargar los. Le collectiones private es displicate in nigrato. Le collectiones es discargate qual glossarios TBX in le plica del currente glossario.

Recerca del terminologia TaaS: quando controlate, permittera querer a TaaS datos sur le base de segmento per segmento. Tote le collectiones (public e private) essera querite pro le linguas fonte e de destination.

Pro limitar le quantitate del datos, il es possibile eliger un dominio specific per Selige le dominio de recerca del terminologia TaaS. In iste fenestra de dialogo, il es possibile eliger Tote le dominios o uno specific.
