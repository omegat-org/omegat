1. Fenestra de recerca

Aperi le Fenestra de recerca conCtrl+F e insere le parola o le phrase tu pro le qual tu desira cercar in le quadro Cercar pro .

O in alternativa elige, in le Fenestra de Modification, un parola o un phrase in le campo de modification (texto de destination del segmento currente) e pulsa Ctrl+F. Le parola o phrase es inserite automaticamente in le quadro Cercar pro Tu pote haber plure Fenestras de recerca aperte al mesme tempore, sed claude los quando illos non son son plus necessari a fin que illos non incombra tu scriptorio.

Clicca le flecha a cadita del cassa Recercar pro al fin de acceder al ultime 10 recercas.

2. Usar characteres generic

In ambe le recercas, exacte e per parola clave, on pote usar le characteres de recerca generic '*' e '?'. Illos ha le significato, familiar al usatores de Word:

  • '*' concorda zero o plus characteres, ab le position actual in un date parola a su fin. Le termino de recerca 'run*' per exemplo facerea concordar le parolas 'run', 'runs' e 'running'.

  • '?' concorda cata singule character. Per exemplo, 'run?' facerea concordar le parola 'runs' e 'runn' in le parola 'running'.

Le concordantias essera marcate in azur hardite. Nota que '*' e '?' ha significato special in le expressiones regular, assi le recerca con character generic, qual describite ci, se refere sol al recerca exacte e al parola clave (vide in successivemente).

3. Methodos de recerca e optiones

Eliger le methodo per le buttones radio. Son disponibile le methodos de recerca sequente:

Recerca exacte

Recerca pro le segmentos continente le linea de character exacte que tu ha specificate. Un recerca exacte quere pro un phrase, id es si plure parolas son entrate, illos son trovate solmente si illos occurre in justo celle sequentia. Usante le recerca con open file trovara assi tote le occurrentias del linea de character open file ma non de file opened o de open input file.

recerca per parola clave

Recerca pro le segmentos continente omne le parolas clave tu specificava, in omne positura. Selige "recerca parola clave" per perquirer pro omne numero de individual parolas complete, in omne ordine. OmegaT monstra un lista de omne segmentos continente tote le parolas specificate. Recercas per parola clave es simile a un recerca "con omne le parolas" in un motor de recerca Interrete como Google (AND logic). Usante le recerca per parola clave con open file trovara assi omne occurrentias del linea de character open file, assi como file opened, open input file, file may not be safe to open, et cetera.

expressiones regular

Le linea de character de recerca essera tractate como un expression regular. Le linea de character de recerca - [a-zA-Z]+[öäüqwß] - in le caso precedente per exemplo appare pro parolas in le segmento de destination, continente characteres dubitabile del claviero german. Le expressiones regular son un maniera potente pro cercar pro occurrentias de un linea de character. Vide plus in le capitulo Expressiones regular.

In addition pro uno del methodos precedente tu pote seliger le sequente:

  • sensibile al majusculas e minusculas: le recerca essera effectuate pro le exacte linea de characteres specificate; id es es observate le capitalisation.

  • Concordantias del spatio nbsp: quando es marcate iste option, un character spatio del fila de recerca concorda sia un normal character de spatio que un character (\u00A) de spatio ininterrumpibile.

  • in le fonte: cerca in le segmentos fonte

  • in le traductiones: cerca in le segmentos de destination

  • in le notas: cerca in le notas del segmentos

  • in le commentos: cerca in le commentos del segmentos

  • Rendite o non rendite: recerca in ambe le segmentos rendite e non rendite.

  • Rendite: recerca sol in le segmentos rendite.

  • Non rendite: recerca sol in le segmentos non rendite.

  • Monstrar: tote le segmentos concordante: si marcate, tote le segmentos es monstrate uno per uno, etsi si illos es presente plure vices in le mesme documento o in documentos differente.

  • Monstrar: le nomines de file: si marcate, le nomine de file ubi cata segmento es trovate, es monstrate ante cata resultato.

  • Cercar in le projecto: marcar Memoria pro includer le memoria de traduction del projecto (le file project_save.tmx) in le recerca. Marcar TMs pro includer in le recerca le memorias de traduction ubicate in le plica tm. Marcar Glossarios pro includer in le recerca le glossarios ubicate in le plica glossary.

  • Recerca in le files: recerca in un file singule o in un plica continente un collection de files. Quando tu cerca trans le files (al contrario que al memorias de traduction), OmegaT restringe le recerca al files in le formatos del file fonte. Per consequentia, ben que OmegaT is bastante capace a tractar files tmx, illo non include los in le Files a perquirer del recerca.

Si tu clicca sur le button Optiones avantiate pote esser seligite altere criterios (autor del traduction, date del traduction et cetera).

4. Monstrar le resultatos del recerca

Pulsante le button de recerca, post haber inscribite in le campo un linea de character, monstra omne le segmentos in le projecto que include le linea de character inserite. Pois que OmegaT tracta segmentos identic como un entitate singule, sol le prime segmento unic es monstrate. Le segmentos es monstrate in ordine de apparition in le projecto. Le segmentos rendite es monstrate con le texto original in alto e le texto rendite in basso, le segmentos non rendite es monstrate con le fonte solmente.

Cliccar sur un segmento aperi lo in le quadro de Modification. Tu pote alora receder al Fenestra de recerca pro trovar le segmento successive, per exemplo pro controlar e, si necesse, corriger le terminologia.

Tu pote haber plure Fenestras de recerca aperte al mesme tempore. Tu pote rapidemente vider lor contentos mirante lor titulo: illo continera le termino de recerca usate.

5. Le entratas de filtro in le editor, conforme al recerca

Pro un navigation plus facile in le resultato de recerca imponite, tu pote applicar le recerca al Quadro de modification. Pulsa le button Filtro in basso pro limitar le entratas monstrate in le Quadro de modification, pro los que concorda le recerca currente. Tu pote usar le navigation normal pro ir per exemplo al segmento (non rendite) successive, que concorda le criterios de recerca.

Notar ben:

  • Un recerca pote esser limitate a 1000 articulos, assi si tu recerca sur un phrase commun, le editor monstra solmente le 1000 entratas concordante, e non omne le entratas que concorda le criterios de recerca.
  • Un file pote haber nulle entratas concordante, assi illo monstrara se vacue.
  • Si un recerca elimina le duplicatos, ille duplicatos non essera in le Editor.

Pro remover un filtro, pulsa le button Remover filtro, o recarga le projecto.
