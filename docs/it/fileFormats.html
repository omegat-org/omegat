Formati di file riconosciuti

OmegaT 1.6.1 è in grado di riconoscere autonomamente i seguenti tipi di file:

  • File OpenDocument (noti anche come documenti OpenOffice.org o StarOffice)
  • File HTML e XHTML ben formati
  • File DocBook
  • File PO
  • File di solo testo
  • File di localizzazione Java (risorse)
  • File INI (“Codice” = “stringa valore”)
  • File Compilatore Help HTML (HCC)

Alcuni interventi manuali consentiranno, inoltre, di tradurre anche file Microsoft Office. Sarà sufficiente utilizzare OpenOffice.org per convertire i file Microsoft Office da e in file OpenDocuments. Per i documenti Microsoft Word, un altro sistema è quello di salvare i file in formato HTML direttamente da Microsoft Word.
