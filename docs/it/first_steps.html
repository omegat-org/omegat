OmegaT 6.0.2

Primi passi

Per aprire il manuale dell'utente premere il tasto F1 o accedere dal menu > Aiuto. Verrà presentata una introduzione accurata su OmegaT. Per iniziare a usare OmegaT con le impostazioni minime:

  1. Usare il comando > Progetto > Nuovo... per creare un nuovo progetto.

  2. Cambiare i codici di lingua affinché corrispondano alle lingue di partenza e di arrivo. Fare clic su OK.

    → OmegaT creerà un gruppo di cartelle con i relativi file. Creerà anche una memoria di traduzione vuota che verrà popolata in sottofondo, un segmento per volta, man mano che si traduce. Le corrispondenze per quella memoria compariranno eventualmente nel pannello delle Concordanze parziali.

  3. Fare clic su Aggiungi file... per aggiungere i file da tradurre. Chiudere la finestra.

    → OmegaT copierà i file selezionati all'interno del progetto creato nel passaggio precedente. I file saranno ora mostrati segmento per segmento in modo da poterli tradurre.

  4. Scrivere la traduzione del primo segmento. Premere Invio.

    → OmegaT registrerà quel segmento all'interno della memoria di traduzione del progetto e sposterà il cursore nel segmento successivo.

  5. Usare, in qualsiasi momento, il comando > Progetto> Crea i file tradotti per creare il documento tradotto.

    → OmegaT memorizzerà i file tradotti all'interno della cartella target del progetto, creata nel primo passaggio. Per accedere ai file, usare il comando > Progetto > Accedi al contenuto del progetto.

  6. Per saperne di più, leggere il manuale dell'utente.
