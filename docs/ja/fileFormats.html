対応するファイル形式

OmegaT 1.6.2 はネイティブに以下のファイルタイプに対応しています:

  • - Microsoft Open XMLファイル形式
  • OpenDocument ファイル (OpenOffice.org または StarOffice ドキュメントとしても知られます)
  • HTML と well-formed XHTML
  • DocBook ファイル
  • PO ファイル
  • プレーンテキストファイル
  • Java各国語対応ファイル(リソースバンドル)
  • INI ファイル (Key=Value 形式のテキスト)
  • HTML Help Compiler ファイル

Microsoft Office 2007以外はファイルの翻訳には多少の手作業が必要です。Microsoft Office ファイルと対応する OpenDocument ファイルの変換に OpenOffice.org を使ってください。Microsoft Word ファイルについては Microsoft Word 自身から HTML で保存するという手もあります。
