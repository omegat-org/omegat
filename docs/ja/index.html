この取り扱い説明ガイドは Jean-Christophe Helary, Samuel Murray, Maxym Mykhalchuk, Henry Pijffers の共同作業の成果です。

Marc Prior が作成して OmegaT version 1.4.2 まで保守し、Dmitri Gabinski が OmegaT version 1.4.4 のために更新した元のドキュメント及び、ユーザーリストからの情報と OmegaT の開発に参加した多くの人々の助けなしには、このようなガイドを一から書くことはできなかったでしょう。OmegaT コミュニティの全ての人々のエネルギーとインスピレーションにも感謝します。

このガイドは完全ではなく、時が経つにつれて、また OmegaT の進化に従って修正されます。もしここに記述されていることと OmegaT の振る舞いとの食い違いに気付いたら、遠慮なく Jean-Christophe Helary ( brandelune AT users DOT sourceforge DOT net)に連絡してください。
