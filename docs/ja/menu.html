メインメニュー

OmegaT の全ての機能はエディタウィンドウの上にあるメニューバーから利用可能です。ほとんどの機能はキーボードショートカット経由でも利用可能です。

Ctrl は Windows, UNIX, UNIX 風のオペレーティングシステムではCtrl / Controlキーを使います。Mac OS X ユーザーは代わりに Apple key を使うことになるでしょう。

プロジェクトメニュー

新規作成
新しいプロジェクトを作成して、開きます。プロジェクトを作成するためのダイアログはプロジェクトの編集のためのものと同じです。
プロジェクトを開く <Ctrl+O>
以前に作成したプロジェクトを開きます。
原文ファイルを追加
選択したファイルを/sourceディレクトリにコピーし、新しいファイルを読み込むためにプロジェクトを再読み込みします。
再読み込み
原文ファイル、参考用翻訳メモリ、用語集の外部での変化を取り込むために、プロジェクトを再読み込みします。
閉じる
翻訳を保存し、プロジェクトを閉じます。
保存 <Ctrl+S>
翻訳をプロジェクトの翻訳メモリに保存します。10 分毎、またはプロジェクトを閉じるか OmegaT を終了させたときに、OmegaT は自動的に翻訳を保存します。
訳文ファイル生成
翻訳を基にした訳文を生成します。翻訳されたファイルは、/targetディレクトリに配置されます。
設定 <Ctrl+E>
プロジェクト言語とフォルダ位置を編集するためのプロジェクト設定ダイアログを表示します。
翻訳対象ファイル一覧... <Ctrl+L>
翻訳対象ファイル一覧ウィンドウを閉じるか開くかします(現在開いているか閉じているかに依存します)。
終了 <Ctrl+Q>
プロジェクトを保存し、OmegaT を終了します。もしプロジェクトを手動で保存していなかった場合、本当に終了していいか確認します。

編集メニュー

ほとんどのアプリケーションで見つけられる機能(コピー/切り取り/貼り付け)はこのメニューには表示されていませんが、システムショートカットを使うことで利用可能です。例えば、 Windows では:

コピー <Ctrl+C>
選択した文章をクリップボードにコピーします。
切り取り <Ctrl+X>
選択した文章をクリップボードにコピーし、選択した文章を削除します。
貼り付け <Ctrl+V>
クリップボードからカーソル位置に文章を貼り付けます。

メニュー自身には以下の項目があります:

操作取り消し &U <Ctrl+Z>
直前に行なった編集動作の前の状態に戻ります。修正された分節が検証されてしまうと動作しません。
操作繰り返し <Ctrl+Y>
直前の編集動作を取り消す前の状態に戻ります。修正された分節が検証されてしまうと動作しません。
参考訳文に置換 <Ctrl+R>
現在注目している分節の内容を選択した参考訳文で置き換えます(デフォルトでは 1 番目の参考訳文が選択されています)
参考訳文を挿入 <Ctrl+I>
カーソル位置に選択した参考訳文を挿入します。この機能は、たとえ文章が選択状態でも、訳文分節は上書きしません。
原文に置換 <Shift+Ctrl+R>
現在注目している分節の内容を原文で置き換えます。
原文を挿入 <Shift+Ctrl+I>
カーソル位置に原文を挿入します。
プロジェクトを検索... <Ctrl+F>
新しい検索ウィンドウを開きます。
参考訳文 #N を選択<Ctrl+N> (N は 1 から 5)
分節に置換または挿入するために、参考訳文ウィンドウに表示されているN番目の参考訳文を選択します。参考訳文の項目に色の意味に関する詳細が記述されています。

移動メニュー

次の未翻訳分節 <Ctrl+U>
翻訳メモリに対応するものがない次の分節に移動します。
次の分節<Ctrl+N>, <Enter> / <Tab>
次の分節に移動します。現在の分節がファイルの最後の場合、次のファイルの最初の分節に移動します。
前の分節<Ctrl+P>, <Ctrl+Enter> / <Ctrl+Tab>
前の分節に移動します。現在の分節がファイルの最初の場合、前のファイルの最後の分節に移動します。

ツールメニュー

タグ検証 <Ctrl+T>
フォーマット化ファイルでタグがなくなったり位置が変わったりしていないかを調べます。タグを間違えている分節の一覧を表示します。タグ検証ウィンドウの項目に、ここで何が表示されているかが説明されていて、フォーマット付きテキストの翻訳でなぜかを説明しています。

設定メニュー

TABで進む
分節検証キーをデフォルトの<Enter>から<Tab>に変更します。このオプションは中国、日本、韓国の文字入力システムで便利です。
フォント...
文書表示用フォントを変更するためのフォント設定ダイアログを表示します。古いコンピューターのユーザーは、プロジェクトを開く前にフォントを変更した方が良いでしょう。このペナルティは G4 + Mac OS X + 日本語の組み合わせのときに最大になります。
ファイルフィルタ...
ファイルの扱いと解析の設定のためのファイルフィルタダイアログを表示します。
分節化...
文書分節化を設定するために翻訳文書分節化ダイアログを表示します。
翻訳入力行...
まだ翻訳されていない分節の訳文フィールドを設定するための翻訳入力行設定ダイアログを表示します。
ウィンドウ初期設定に戻す
メイン OmegaT ウィンドウのコンポーネント(エディタ、参考訳文、用語集)をデフォルトの状態に戻します。これらのコンポーネントを結合解除したり移動したり他のコンポーネントで隠したりして、元の状態に戻せなくなったときにこの機能を使ってください。

ヘルプメニュー

取り扱い説明ガイド... <F1>
このガイドを別のウィンドウで表示する為のヘルプブラウザを開きます。
OmegaTについて...
著作権、クレジット、ライセンス情報を表示します。
