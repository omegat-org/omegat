Настройки шрифтов

Это диалоговое окно позволяет вам изменить шрифт, который OmegaT использует в следующих окнах:

  • Главное окно OmegaT (редактор, панель просмотра совпадений, панель глоссария)
  • Окно поиска
  • Окно проверки тэгов

Его можно вызвать через пункт Опции → Шрифт... в Главном меню.

Это диалоговое окно содержит:

  • Меню выбора одного из шрифтов, доступных на вашем компьютере
  • Поле изменения размера шрифта
  • Область предварительного просмотра выбранного шрифта

Внимание! OmegaT может потребоваться довольно значительное время, чтобы обновить окна после изменения настроек шрифта, особенно если в редакторе открыт большой файл со множеством сегментов. Кроме того, некоторым языковым парам больше подходят определённые шрифты.
