如何下载并安装词典

OmegaT 中的词典基于 StarDict 或 Lingvo DSL 格式。要安装 StarDict 格式所必需的文件，请按以下步骤操作：

  • 搜索所需的语言组合——例如通过 OmegaT Wiki 给出的词典链接

  • 下载文件——它应当是个 tarball 存档（扩展名为 tar.bz 或 tar.bz2 ）。

  • 用 untar 工具（或等效物，比如 Windows 中的 winrar）将其内容提取到项目文件夹“Dictionary”中。应当有三个文件，扩展名分别为 dz、idx 和 ifo 。

请注意，除了“原文-译文”类型的词典之外，你还可以通过词典功能访问以下信息：

  • 韦氏国际词典（Webster's Revised Unabridged Dictionary）（1913）

  • 朗文当代英语词典（Longman Dictionary of Contemporary English）

  • 大英简明百科（The Britannica Concise Encyclopedia）

  • 等等

与词典相关的问题

  • 检查词典文件是否在正确的文件夹中（或在其下面的子文件夹中）。在 项目 → 属性 ( Ctrl + E ) 中检查。

  • 该文件夹是否包含三个带不同扩展名的同名文件？如果只有一个文件，请检查其扩展名。如果是 tar.bz ，那么意味着你忘记把它解压缩（untar）了。
