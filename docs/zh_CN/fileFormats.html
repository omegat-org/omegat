支持的文件格式

OmegaT 1.6.1 先天支持以下文件类型：

  • 开放文档文件（也称为 OpenOffice.org 和 StarOffice 文档）
  • HTML 和合式的 XHTML
  • DocBook 文件
  • PO 文件
  • 纯文本文件
  • Java 本地化文件（资源包）
  • INI 文件（Key=Value 文本）
  • HTML 帮助编译器文件

通过一点手工处理，你也可以翻译 Microsoft Office 文件。只需要使用 OpenOffice.org 将文件在 Microsoft Office 格式和对应的开放文档格式之间来回转换。对于 Microsoft Word 文件，还有一个办法是使用微软的 Word 软件将它们存为 HTML 文件。
