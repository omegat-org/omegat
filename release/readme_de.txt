Was ist OmegaT?
===============

OmegaT ist ein kostenloses und quelloffenes Multiplattform-Tool für
computergestützte Übersetzung mit Fuzzy-Matching, Translation Memory,
Stichwortsuche, Glossaren und Einbindung von Übersetzungen in aktualisierte
Projekte.

Voraussetzungen für die OmegaT-Installation
===========================================

Für OmegaT muss Java Runtime Environment (JRE) Version 1.8 oder höher
auf Ihrem System installiert sein.

Wir empfehlen die Verwendung von OmegaT-Paketen, die die JRE bereits enthalten,
um Ihnen die Suche, Beschaffung und Installation der JRE zu ersparen.



OmegaT installieren (Windows)
=============================

Starten Sie das Installationsprogramm.



OmegaT installieren (Mac)
=========================

Entpacken Sie das .zip-Archiv von OmegaT, um einen Ordner mit einer
Dokumentationsdatei und der OmegaT-Anwendung zu erhalten. Verschieben Sie den
Ordner an einen geeigneten Ort, z.B. in den Programme-Ordner.



OmegaT installieren (Linux)
===========================

Platzieren Sie das Archiv in einen beliebigen Ordner und entpacken Sie es. 
Danach kann OmegaT gestartet werden.

Für eine übersichtlichere und benutzerfreundlichere Installation können Sie das
Installationsskript (linux-install.sh) verwenden. Um dieses Skript zu
verwenden, öffnen Sie ein Terminal-Fenster (Konsole), wechseln Sie zum Ordner,
der OmegaT.jar und das Skript linux-install.sh enthält, und führen Sie das
Skript mit ./linux-install.sh aus.



OmegaT installieren (Solaris, FreeBSD, etc.)
============================================

Platzieren Sie das Archiv in einen beliebigen Ordner und entpacken Sie es. 
Danach kann OmegaT gestartet werden.



OmegaT starten (Windows)
========================

Wenn Sie während der Installation eine Verknüpfung auf dem Desktop erstellt
haben, doppelklicken Sie auf diese Verknüpfung.

Das Installationsprogramm kann für Sie Verknüpfungen im Startmenü, auf dem
Desktop und im Schnellstartbereich erstellen. Sie können die Datei OmegaT.exe
auch manuell in das Startmenü, auf den Desktop oder in den Schnellstartbereich
ziehen, um sie von dort aus zu verknüpfen.

Wenn Sie in Ihrem Dateimanager (Windows Explorer) die Datei OmegaT, aber nicht
OmegaT.exe sehen können, ändern Sie die Einstellungen so, dass Dateiendungen
angezeigt werden.



OmegaT starten (Mac)
====================

Doppelklicken Sie auf das Programm OmegaT.

Sie können das OmegaT-Programm auf das Dock oder in die Symbolleiste eines
Finder-Fensters ziehen, um es von jedem Ort aus starten zu können. Sie können
es auch über das Spotlight-Suchfeld starten.



OmegaT starten (Linux)
======================

Wenn Sie das Skript linux-install.sh verwendet haben, sollten Sie OmegaT wie
folgt starten können:

    Alt+F2

und danach:

    omegat

Um OmegaT benutzerfreundlicher zu starten, können Sie das im Lieferumfang
enthaltene Kaptain-Skript verwenden (omegat.kaptn). Um dieses Skript zu
verwenden, müssen Sie zunächst Kaptain installieren. Sie können dann das
Kaptain-Startskript wie folgt starten:

    Alt+F2

und danach:

    omegat.kaptn



OmegaT von der Befehlszeile aus starten (alle Systeme)
======================================================

Der Befehl zum Starten von OmegaT lautet:

    cd <Ordner mit der Datei OmegaT.jar>

    <Name und Pfad der Java-Executable-Datei> -jar OmegaT.jar

(Die Java-Executable-Datei ist in Linux die Datei java und in Windows die Datei
java.exe.  Wenn Java auf Systemebene installiert ist und sich im Befehlspfad
befindet, muss der vollständige Pfad nicht eingegeben werden.)
