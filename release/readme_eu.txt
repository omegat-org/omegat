Zer da OmegaT?
===============

OmegaT aplikazioa Ordenagailuz Lagundutako Itzulpenetarako tresna da, plataforma anitzetarako prestatua, kode irekikoa, parekatze lausoz, itzulpen-memoriaz, gako-hitzen bidezko bilaketaz eta glosarioz hornitutakoa, eta egindako itzulpenak proiektu eguneratuetan aprobetxatzea ahalbidetzen duena.

OmegaT instalatu aurreko baldintzak
===================================

OmegaT-k Java Runtime Environment (JRE) 1.8 edo berriagoa behar du zure sisteman instalatu ahal izateko.

JRE integratuta daukaten OmegaT paketeak erabiltzea gomendatzen dugu, arazorik izan ez dezazun hura hautatu, eskuratu eta instalatzeko orduan.



OmegaT-ren instalazioa (Windows)
===========================

Abiarazi instalazio-programa



OmegaT-ren instalazioa (Mac)
===========================

Deskonprimitu OmegaT .zip fitxategia, aplikazioa eta dokumentazio-fitxategiak dituen karpeta bat erauzteko. Mugitu karpeta kokaleku egoki batera, adibidez aplikazioen karpetara.



Installing OmegaT-ren instalazioa (Linux)
=========================

Jarri fitxategia edozein karpeta egokitan eta deskonprimitu. Horren ondoren, OmegaT prest dago abiarazia izateko.

Hala ere, instalazio txukunago eta lagungarriagoa egin dezakezu instalazio-scripta erabilita (linux-install.sh). Script hori erabiltzeko, ireki terminala (kontsola,
joan OmegaT.jar eta linux-install.sh scripta dauden karpetara
eta exekutatu hura ./linux-install.sh erabilita. 



OmegaT-ren instalazioa (Solaris, FreeBSD, ...)
===========================

Jarri fitxategia edozein karpeta egokitan eta deskonprimitu. Horren ondoren, OmegaT prest dago abiarazia izateko.



OmegaT-ren instalazioa Java Web Start erabilita (plataforma guztiak)
===========================================================

Jadanik Java badaukazu sisteman instalatuta, OmegaT-ren bertsiorik berriena instalatzeko modu bat Java Web Start erabiltzea da.

Horretarako, deskargatu hurrengo fitxategia eta exekuta ezazu:

   http://omegat.sourceforge.net/webstart/OmegaT.jnlp

Zure ordenagailuak behar duen ingurune egokia eta aplikazioa bera instalatuko du. Geroagoko abioetan ez da beharrezkoa izango linean egotea. 



OmegaT abiaraztea (Windows)
==========================

Instalazioan zehar mahaigainean lasterbide bat sortu baduzue, egin klik bikoitza haren gainean.

Instalazio-programak lasterbideak sor ditzake hasiera-menuan, mahaigainean eta abiarazte bizkorreko arean. OmegaT.exe fitxategia mahaigainera edo hasiera-menura arrastatu daiteke aplikazioa bertatik abiarazteko.

Zure fitxategi-kudeatzailean (Windows Explorer) OmegaT fitxategia ikus badezakezu, eta ez OmegaT.exe fitxategia, aldatu Windows-en hobespenak fitxategien luzapenak erakutsi ditzan.



OmegaT abiaraztea (Mac)
==========================

Egin klik bikoitza OmeagT aplikazioan.

OmegaT aplikazioa zuen dock-era edo Finder leiho baten tresna-barrara arrastatu dezakezue, hura edozein kokapenetik abiarazi ahal izateko. Spotlight-en bilaketa-kutxatik
ere abiarazi dezakezue.



OmegaT abiaraztea (Linux)
==========================

linux-install.sh scripta erabili baduzu, OmegaT honela abiarazi dezakezu:

  Alt+F2

eta ondoren:

  omegat

OmegaT abiarazteko modu errazagoa edukitzeko, Kaptain scripta erabil dezakezue (omegat.kaptn). Script hau erabiltzeko, lehenago Kaptain instalatu behar duzu. Ondoren, Kaptain abio-script-a abiaraz dezakezue honela:

  Alt+F2

eta ondoren:

  omegat.kaptn



OmegaT komando-lerrotik abiaraztea (sistema guztiak)
====================================================

OmegaT abiarazteko komandoa honakoa da:

cd <OmegaT.jar kokatuta dagoen karpeta>

<Java fitxategi exekutagarriaren izena eta bidea> -jar OmegaT.jar

(Java fitxategi exekutagarria java.exe deitzen da Windows-en eta java Linux-en.
Java sistema mailan instalatuta badago eta komando-bidean badago, ez da beharrezkoa bide osoa sartzea.)
