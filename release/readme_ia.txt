Quod es OmegaT?
===============

OmegaT is a free and open source multiplatform Computer Assisted Translation tool with fuzzy matching, translation memory, keyword search, glossaries, and translation leveraging into updated projects.
(OmegaT es un application gratuite de Traduction Assistite per le Computator (CAT), multe-platteforma con concordantia partial, recerca per parola clave, glossarios, e fruition del traductiones in le actualisation del projectos)

Pre-requisitos pro installar OmegaT
===================================

Pro installar OmegaT sur tu systema es necesse Java Runtime Environment (JRE), version 1.8 o plus recente.

Nos recommenda de usar le pacchettos de OmegaT que include le JRE, pro te sparniar del
enoio de lo selectionar, obtener e installar.



Installar OmegaT (Windows)
===========================

Lancea le programma de installation.



Installar OmegaT (Mac)
=======================

Discassa le archivo OmegaT.zip pro obtener un plica que contine un
file de documentation e le application OmegaT. Move le plica a un
position appropriate como le plica Applicationes.



Installar OmegaT (Linux)
=========================

Poner le archivo in qualcunque convenibile plica e discassa illo. OmegaT es jam preste
a esser lanceate.

Comocunque tu pote obtener un installation plus nette e plus facile per medio del script de installation (linux-install.sh). Pro usar ce script, aperi un 
fenestra terminal (consola), cambia le plica in le plica
continente le scriptes OmegaT.jar e linux-install.sh, e exeque le
script ./linux-install.sh.



Installar OmegaT (Solaris, FreeBSD, ...)
=========================================

Poner le archivo in qualcunque convenibile plica e discassa illo. OmegaT es jam preste
a esser lanceate.



Usar Java Web Start pro installar OmegaT (tote le platteformas)
===========================================================

Si tu ha jam Java installate sur tu Systema, un maniera pro installar OmegaT 
es usar Java Web Start..

Pro ce fin, discarga le sequente file e pois exeque lo:

   http://omegat.sourceforge.net/webstart/OmegaT.jnlp

Illo installara le correcte ambiente pro tu computator e le application
ipse al prime lanceamento. Le successive executiones non habe necessitate de esser in linea.



Como lancear OmegaT (Windows)
==========================

Si, durante le installation, tu ha create un via breve sur le scriptorio, duple-
clicca sur ille via breve.

Le programma de installation pote crear vias breve pro te in le menu de initio, sur le
scriptorio e in le area de lanceamento celere. Tu pote alsi manualmente traher le file
OmegaT.exe al menu de initio, al scriptorio o al area de lanceamento celere pro lo ligar ab ibi.

Tu pote vider le file OmegaT, ma non OmegaT.exe in tu 
gestor de files (Windows Explorer), cambia le configurationes assi que le extensiones del files 
es monstrate.



Como lancear OmegaT (Mac)
======================

Clicca duo vices sur le application OmegaT.

Tu pote traher le application OmegaT a tu bassino o al barra de applicationes del un
Fenestra trovator pro poter lo lancear ab qualcunque position. Tu pote anque lancear lo in le 
campo de recerca de Spotlight.



Como lancear OmegaT (Linux)
========================

Si tu usava le script linux-install.sh, tu potera lancear OmegaT con:

  Alt+F2

e deinde:

  omegat

Pro un modo plus facile de lancear OmegaT, tu pote usar le 
script Kaptain (omegat.kaptn) providite. A usar iste script tu debe primo installar
Kaptain. Deinde tu pote lancear le script de lanceamento de Kaptain per:

  Alt+F2

e deinde:

  omegat.kaptn



Lancear OmegaT per linea de commando (tote le systemas)
====================================================

Le commando pro lancear OmegaT es:

cd <plica ubi le file OmegaT.jar es localisate>

<nomine e itinere del file executabile Java> -jar OmegaT.jar

(Le file executabile Java es le file java sur Linux e java.exe sur Windows.
Si Java es installate a nivello de systema e es in le percurso del commando,  
non necessita inserer le percurso integral.)
