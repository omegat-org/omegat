Vad är OmegaT?
===============

OmegaT är ett fritt multiplattformsverktyg för datorstödd översättning baserat på öppen
källkod som har suddig matchning, översättningsminne, nyckelordssökning, gloslistor
och översättningshantering till uppdaterade projekt.

Förutsättningar för att installera OmegaT
===================================

OmegaT kräver Java Runtime Environment (JRE) version 1.8 eller högre för att kunna
installeras i ditt system.

Vi rekommenderar att man använder ett OmegaT-paket som redan innehåller JRE, för
att bespara sig besväret med att välja, ladda ner och installera den.



Installera OmegaT (Windows)
===========================

Starta installationsprogrammet.



Installera OmegaT (Mac)
=======================

Packa upp OmegaT.zip-filen för att få en mapp som innehåller
en fil med dokumentation samt OmegaT-programmet. Flytta mappen till en
lämplig plats såsom Programmappen.



Installera OmegaT (Linux)
=========================

Placera arkivet i någon lämplig mapp och packa upp det. OmegaT är sedan färdigt
att startas.

Men du kan få en snyggare och mer användarvänlig installation genom
att använda installations-skriptet (linux-install.sh). För att använda detta skript öppnar du ett
terminal-fönster (konsol), ändrar aktuell mapp till mappen som innehåller OmegaT.jar
och skriptet linux-install.sh, och kör skriptet med ./linux-install.sh.



Installera OmegaT (Solaris, FreeBSD, …)
=========================================

Placera arkivet i någon lämplig mapp och packa upp det. OmegaT är sedan färdigt
att startas.



Använda Java Web Start för att installera OmegaT (alla plattformar)
===========================================================

Om du redan har Java installerat i ditt system, är Java Web Start ett sätt att
installera OmegaT.

För detta syfte, ladda ner följande fil och kör den:

   https://omegat.sourceforge.io/webstart/OmegaT.jnlp

Det kommer att installera rätt miljö för din dator och själva programmet
vid första körningen. Senare anrop behöver inte göras online.



Starta OmegaT (Windows)
==========================

Om du skapade en genväg på skrivbordet vid installationen, dubbelklicka
på den genvägen.

Installationsprogrammet kan skapa genvägar åt dig i start-menyn, på skrivbordet
och i snabbstart-området. Du kan också manuellt dra filen
OmegaT.exe till Start-menyn, skrivbordet eller snabbstart-området för att länka det
därifrån.

Om du kan se filen OmegaT men inte OmegaT.exe i Utforskaren
(Windows Explorer), ska du ändra inställningarna så att filnamnstillägg visas.



Starta OmegaT (Mac)
======================

Dubbelklicka på programmet OmegaT.

Du kan dra programmet OmegaT till din Dock eller verktygslisten på ett
Finder-fönster för att kunna starta det från valfri plats. Du kan också starta det
från sökfältet Spotlight.



Starta OmegaT (Linux)
========================

Om du använde skriptet linux-install.sh, ska du kunna starta OmegaT med:

  Alt+F2

och sedan:

  omegat

För ett mer användarvänligt sätt att starta OmegaT kan du använda medföljande Kaptain
skript (omegat.kaptn). För att använda detta skript måste du först installera
Kaptain. Sedan kan du köra Kaptain-startskriptet med:

  Alt+F2

och sedan:

  omegat.kaptn



Starta OmegaT från kommandoraden (alla system)
==========================================

Kommandot för att starta OmegaT är:

cd <mappen där filen OmegaT.jar finns>

<namn och sökväg till den körbara Java-filen> -jar OmegaT.jar

(Den körbara Java-filen är filen java i Linux och java.exe i Windows.
Om Java är installerat på systemnivå och finns i kommandosökvägen behöver inte
den fullständiga sökvägen anges.)
