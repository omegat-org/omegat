Creació de les vostres pròpies memòries de traducció

Podeu exportar un fitxer TMX de cada projecte de l'OmegaT per a lliurar-lo i com a memòria de traducció de referència d'un projecte futur.

L'OmegaT exportarà les memòries quan seleccioneu Projecte Crea els fitxers traduïts i compartirà les memòries de traducció exportades quan sigui necessari.
