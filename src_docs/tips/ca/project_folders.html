Carpetes del projecte

Un projecte de l'OmegaT està format per un conjunt de carpetes i fitxers que contenen tots els recursos que s'utilitzen per a traduir. Els noms per defecte són els següents.

  • «source» conté els fitxers d'origen
  • «target» conté els fitxers de destinació creats
  • «glossary» conté els glossaris
  • «glossary.txt» és el glossari modificable del projecte
  • «tm» conté les memòries de traducció de referència
  • «dictionary» conté els diccionaris de referència
