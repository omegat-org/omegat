Projektordner

Ein OmegaT-Projekt besteht aus einer Reihe von Ordnern und Dateien, die Ressourcen für die Übersetzung enthalten. The default names are as follows.

  • source enthält die Quelldateien
  • target is the location where target files are created
  • glossary enthält die Glossare
  • tm enthält die externen Translation Memories
  • dictionary enthält die externen Wörterbücher
glossary.txt is the project writable glossary.
