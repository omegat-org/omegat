Dictionnaires

OmegaT prend en charge les dictionnaires au format StarDict ou Lingvo DSL. You can find some dictionaries on the OmegaT Wiki.

Pour installer les dictionnaires :

  1. download the file - it should be a tarball archive (extension tar.bz or tar.bz2),

  2. extrayez son contenu dans ce dossier. There should be three files per dictionary, with extensions dz, idx and ifo.
