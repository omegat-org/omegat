Tema di OmegaT

È possibile selezionare un tema per l'interfaccia utente di OmegaT.

È possibile aggiungere temi aggiuntivi come plugin. I temi forniscono anche un set predefinito di colori predefiniti dei contrassegni.

Il tema standard di OmegaT passerà ai colori predefiniti basati sui colori di sistema.
