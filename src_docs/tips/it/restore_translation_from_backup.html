Ripristino di traduzioni perse

Se pensi che il tuo lavoro possa andare perduto, chiudi il progetto e usa un sistema di copie di sicurezza automatico del file project_save.tmx nella cartella omegat del progetto.

Rinomina una copia adatta (per es., project_save.tmx.YYYYMMDDhhmm.bak) in project_save.tmx e riapri il progetto per ripristinare quello stato.
